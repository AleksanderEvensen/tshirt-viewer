# T-Shirt Viewer

Interactive 3D t-shirt designer — drop images onto the design canvas and see them rendered on the model in real time.

**Live demo:** https://aleksanderevensen.github.io/tshirt-viewer/

Built with Three.js, Fabric.js, Vite, and TypeScript.

## Development

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```

Deploys automatically to GitHub Pages via `.github/workflows/deploy.yml` on every push to `main`.
